# MineNBTool
这是一款 Minecraft 附魔指令生成器，它可以方便、快捷的生成 Minecraft Java Edition 1.7.2-1.20.2 的附魔指令。

**立即使用** [GhPages](https://lagsnesowo.github.io/MineNBTool) | [CfPages](https://minenbtool.lagsn.es)

## 特点
 - **DeepSeek-V3 大模型 AI 一键生成指令，方便快捷**
 - UI 界面舒适，适合普通用户
 - 操作方便快捷
 - 可自定义物品 NBT 名称、颜色
 - 种类多样，易于添加
